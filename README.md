# Student Management System

## Description

A simple Student Management System developed using **HTML, CSS, and JavaScript**. This project allows users to enter student details through a form and dynamically display them in a table. Users can also delete student records using a Delete button.

## Features

* Add student details dynamically
* Delete student records
* Dynamic table generation using DOM Manipulation
* User-friendly form interface
* Responsive and clean UI with CSS

## Technologies Used

* HTML
* CSS
* JavaScript

## Concepts Practiced

* DOM Manipulation
* Dynamic Element Creation
* Event Handling
* Forms and Tables
* Input Validation
* JavaScript Functions

## Future Improvements

* Edit student records
* Search student details
* Store data using Local Storage
* Form validation for empty fields
